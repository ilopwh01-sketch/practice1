# 260426_한화큐셀_업무자동화

한화큐셀 업무자동화 실습을 위한 개인 연습 저장소입니다.

현재 저장소는 GitHub 연동 확인과 작업 기록 관리를 위한 초기 상태이며, 이후 셀 공정 데이터 분석, 수율 개선 리포트, 자동화 스크립트, HTML 요약 화면 등을 정리하는 용도로 사용할 예정입니다.

## 목적

- 배터리/셀 공정 데이터 분석 흐름 연습
- 일자별 생산 CSV 데이터 기반 수율, NG율, 불량 유형 분석
- 라인, 설비, 제품, 교대조, 작업팀 기준의 이상 신호 확인
- 분석 결과를 Markdown/HTML 리포트로 정리
- 반복 분석 작업을 자동화하는 포트폴리오용 실습

## 현재 분석 예시

- 분석 기간: 2026-02-01 ~ 2026-02-10
- 분석 단위: 완성 셀 1개당 1행
- 전체 검사 셀: 5,328개
- 전체 수율: 97.80%
- 전체 NG율: 2.20%
- 주요 개선 포인트: `COSMETIC` 불량, `L03` 설비군, `EQ-L03-03`

## 주요 산출물 예정

- `analysis_report.md`: 수율 개선 관점의 분석 리포트
- `analysis_summary.html`: 브라우저에서 확인할 수 있는 요약 리포트
- `daily_ng_rate.csv`: 일자별 NG율 집계
- `defect_pareto.csv`: 불량 코드 파레토 분석
- `line_ng_rate.csv`: 라인별 NG율
- `equipment_ng_rate.csv`: 설비별 NG율
- `boxplot_stats.csv`: 공정 지표별 분포 비교

## 사용 방향

1. 일자별 CSV 데이터를 준비한다.
2. 수율, NG율, 불량코드, 설비별 집중도를 집계한다.
3. 개선 우선순위를 정리한다.
4. Markdown과 HTML 형태로 결과를 공유한다.
5. 반복 분석이 가능하도록 자동화한다.

## 메모

이 저장소의 데이터와 분석 내용은 실습용 더미 데이터를 기반으로 하며, 실제 회사 내부 데이터가 아닙니다.
